# GameCreator

## 如何使用

1. 安装git (windows下安装git)

2. 使用git bash: $ git clone https://github.com/beckheng/GameCreator

3. $ cd GameCreator

4. $ perl scripts/project_create.pl "D:\MyDemo" BBTowerDefense

5. 使用Unity打开工程: D:\MyDemo\BBTowerDefense\BBTowerDefense_Client
